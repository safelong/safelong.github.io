safelong.github.io
==================

本博客在掌心基础上进行修改。


元主题的介绍位置在[http://www.zhanxin.info/jekyll/2014-01-04-another-jekyll-theme-ilotus.html](http://www.zhanxin.info/jekyll/2014-01-04-another-jekyll-theme-ilotus.html)

修改了以下几点

* 1.首页增加了翻页功能
* 2.分类页面实现了按照类别分类，以前是按照年份
* 3.关于我的图标进行了替换，并且增加了页面
* 4.修改页面双排显示，变成一列显示
* 5.增加了highlight.css，实现了代码高亮
* 6.修改style.css，调整了一些现实效果和图片的自动缩放
* 7.增加了多说评论插件
* 8.增加了百度统计代码
* 9.整体手机端现实正常
