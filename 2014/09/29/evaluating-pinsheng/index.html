品胜云座·易充随身路由器评测

有幸参加了品胜官网组织的云座易充的万台免费试用！

由于本地仓库和附近城市的仓库没有货了，只能从遥远的广东发货了。

本以为需要很多天，没想到，2天就拿到了。

经过简单的试用，现在把自己的不成熟评测分享给大家。

详细参数

云座易充——小巧便携的随身路由器，尽享移动互联

品胜 云座•易充5000mAh移动电源/无线存储二合一设计，尽显前沿科技，支持U盘,SD卡，无线路由、有限无线均可使用。移动存储电能，可随身携带，室内、户外均可使用。

  • 品名：云座·易充
  • 输入：5V-1A
  • 输出：USB 5V-1A
  • 体积：107.951.230.3mm
  • 重量：149g
  • 容量：5000mAh
  • 颜色：苹果白
  • IP地址：192.168.222.254
  • 内置电芯充满电时间：5V-1A充电器，需5-6个小时
  • 价格：价值210元

配件

使用及设置

主界面

把云端易充正面的开关调至WIFI开关，不一会手机就能找到WIFI信号，连上（第一次没有密码，建议先设定密码），打开手机的浏览器，输192.168.222.254之后就可以直接看到主界面了

菜单-文件管理

在这里可以管理上传到云端（路由器SD卡/U盘里）的文件（后面有SD卡/U盘的详细介绍）

菜单-基本设置

这里可以直接设定WIFI热点的名称和密码，建议第一次使用先在这里设定好密码（设定后需重新连接云端易充)。

菜单-外网接入

这里默认是有线路由，在一些没有WIFI的酒店，可以通过提供的有线网线插入云端易充的网口直接将有线信号转化为无线信号，当然其他两个选择也可以按需要选择。

菜单-高级设置

普通用户一般用不到，这里也不介绍了.看看图得了！

菜单-固件升级

可以从官网上下载好最新的固件存手机上给云端易充进行升级，查了，目前的1.5.10已经是最新版了。希望品胜的工程师们，定期的进行固件升级，使该产品越来越强大！

无线路由

上面介绍过，外网接入有三种方式：

  1. 无线：连接到附近能搜索到的wifi信号，和手机连接wifi的操作基本一样
  2. 有线：直接在网口插入网线，和我们安装无线路由是一样的道理
  3. 3G：这个就是所谓的接入3G上网卡

具体怎么配置和使用，我就不详细说了，大家可以自己摸索。

反正，通过以上任何一种方式接入，我们的云座就变成了一个无线路由了。这个功能在出差时用处还是比较大的。

结束总结

一个价值210元的小家伙，集无线路由、移动电源、云存储为一身，性价比还是很高的。

使用技巧：

  1. 开关拨到wifi挡，可以边用路由，边给手机充电
  2. 开关拨到wifi挡，用数据线连接电脑，可以作为读卡器使用
  3. 用re管理器或者Wireless Cloud，可以实现云座和手机端的文件互相传输
  4. 移动电源之间，可以互相充电
  5. 可以外接移动硬盘，我的是西数500G硬盘

最后再说说自己不满意点和建议吧。

  1. 作为移动电源，容量有点小，再扩大2500毫安就很好了。当然，那样的话，体积又变大了
  2. 自带APP的文件管理功能还不够强大，希望能在云座和手机之间，自由复制，如同手机里的存储卡一样。推荐使用：Wireless Cloud
  3. 同时接入两个外接存储，不能够识别。要能同时管理两个外接设备，并且两个外接设备之间可以互传就完美了。
  4. 外接SD卡，SD卡会有很小部分在外面，可能由于弧度的关系，应该可以避免
  5. 3G网卡这个功能对我来说多余，可以去除，降低成本。而且，3G网卡型号很多，兼容性不一定好，可能成为鸡肋功能。
  6. 自身要是能够带一些存储空间就好多了，比如1G空间，就能够缓解没有外接存储时的尴尬局面。
  7. 手机端APP，要是能够自动上传指定目录的图片就行了，相当于定时备份
  8. 没有充电头和保护套，也没多大的成本。要是有的话，用户满意度会很高
  9. 还有个疑问：边充电，边使用无线路由功能，有木有影响？任何充电头都可以充电吗？

以上评测，纯属从自己的需求出发。不求全面，只求实用。
