上传第三方jar包到SonaType Nexus

maven给我们构建工程，提供了极大的方便。

所有引用的jar包，通过maven的pom文件就能完全管理了，免去了我们一个一个去下载的过程。

但是，我们使用maven的过程中，也经常发现一些jar下载不到。

或者，我们公司内部的jar包，不能通过这种pom文件来管理。

当然，每个开发人员，可以自己手动引进jar包或者手动把该jar包放在自己本地仓库。

这样的话，就不能集中管理了。
