homeassistant及插件商店HACS安装

发表于 2023-03-01 更新于 2023-03-02 分类于 技术
HomeAssistant，简称 HA，是一款基于 Python 的智能家居开源系统，支持众多品牌的智能家居设备，可以轻松实现设备的语音控制、自动化等。 HACS是homeassistant的前端商店，就是主要用于页面美化以及个性化自定义的。HACA商店里面有小米的插件Xiaomi Miot Auto

前提

家里小米设备比较多，现在都是通过米家app来控制。由于使用的是远程网关，所以当网络断了，或者小米服务器挂了，米家app就失效了。小米也推出了本地网关，但是价格太贵。正好手上有移动魔百盒，也刷成了armbian系统，所以就想搭建一个本地网关，达到控制小米设备的目的。

安装home assistant

直接用armbian软件中心安装，全程自动，默认是docker安装
软件中心说明：
https://github.com/ophub/amlogic-s9xxx-armbian/blob/main/build-armbian/documents/armbian_software.md

安装完成后，就会出现一个名为“ homeassistant”的容器，可以通过容器管理器portainer来查看和管理容器和镜像

docker安装默认的似乎host网络，这时就可以通过网页来访问homeassistant了。如果是桥接，就需要配置端口映射。

安装HACS

第一步完成后，会发现以下问题

  • 配置-集成，里面并搜不到”Xiaomi Miot Auto“
  • Home Assistant的首页侧边栏并没有HACA

所以，这就需要我们安装HACA，目前网上的安装教程，基本走不下去，不是网络不通，就是手动安装很麻烦。
经过好几天的寻找，下面的教程可以用一行命令解决问题，

HACS是一款优秀的 Home Assistant 集成商店，然而国人想要使用它下载插件或前端卡片却困难重重，主要原因就是国内的网络环境。 本项目使用了ghproxy.com和fastgit.org提供的Github镜像服务，可以让大家更快的下载商店里的插件。

使用命令行安装(推荐)

**手动安装

安装成功后，侧边栏就会出现HACS，点击去就可以搜索Xiaomi Miot Auto并安装了

# docker # armbian # home assistant # hacs
