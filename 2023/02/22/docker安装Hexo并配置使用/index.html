docker安装Hexo并配置使用

发表于 2023-02-22 更新于 2023-02-24 分类于 技术
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。docker安装，也免去自己搭建node.js的烦恼。

前提

自从用魔百盒刷了armbian系统作为一个轻量服务器，一直想部署一个自己博客。之前博客是使用github pages搭建，无需购买服务器。但是github经常抽风，随时可能连接不上。考虑自己写文章的概率不是很大，所以想寻找一个轻量级的开源框架，无需动态，不要数据库，静态页面就行。经过比较，选定了hexo。Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。


又不想破坏armbian系统，所以又首选docker安装，也免去自己搭建node.js的烦恼。几经比较，选定了bloodstar/hexo。这个镜像更新比较频繁，文档说明也很全面，基本是傻瓜教程。我再提炼一下重点，方便更快的入手。


参考:https://hub.docker.com/r/bloodstar/hexo

安装hexo主题

更换 Hexo 主题非常容易，您只要在 themes 文件夹内，新增选定主题的文件夹，并修改 _config.yml 内的 theme 设定，即可切换主题。

主题选择清单：https://github.com/hexojs/hexo/wiki/Themes#
根据自己的喜好，选择合适的主题。github上也有很多非官方的主题，选择多多
每一个主题都有对应网站可以预览，并且都有 Github 地址，我们可以照着 Github 的 wiki 就可以安装了，过程都比较简单。

比如next主题的说明文档：https://theme-next.js.org/docs/getting-started/
里面有详细的配置说明，可以定制个性化的网站页面
